## 💻 技术架构

### 技术栈详情
```
前端层
├── Next.js 14 (App Router)
├── React 18
└── TypeScript 5

样式层
├── Tailwind CSS 3
├── CSS Modules
└── PostCSS

Canvas 层
├── Fabric.js 5.3.0
└── 自定义工具函数

状态管理
├── Zustand 4.4.7
├── React Hooks
└── Context API

工具层
├── Jest 29 (测试)
├── ESLint (代码检查)
├── Prettier (格式化)
└── Husky (Git Hooks)
```

### 项目结构
```
monorepo
├── apps/
│   └── web/              # Next.js 主应用
│       ├── src/
│       │   ├── app/       # App Router 页面
│       │   ├── components/# React 组件
│       │   ├── lib/       # 工具库
│       │   ├── hooks/     # 自定义 Hook
│       │   └── stores/    # 状态管理
│       └── tests/         # 测试文件
├── packages/
│   └── shared-types/     # 共享类型
└── docs/                # 项目文档
```
