# 6. 核心功能路线图 (Core Feature Roadmap)

阶段 1: MVP (核心画布功能)

画布系统： 支持自定义尺寸 (3:4, 1:1, 4:3) 和纯色背景。

文字工具： 支持添加/编辑文字，集成10种基础免费字体。

图片支持： 支持本地图片上传和在画布上进行基本操作（缩放、旋转）。

基础图形： 支持快捷键绘制矩形、圆形和直线。

导出功能： 支持将画布内容复制到剪贴板和下载为 PNG 图片。

阶段 2: 素材库与高级编辑

高级字体： 集成"中文网字计划"和"fonts.zeoseven.com"的精选字体（例如18款）。

高级背景： 支持渐变色背景（集成 gradienthunt.com 灵感，支持"马卡龙色"生成）。

图标与Emoji： 集成 Lucide 图标库（约250个）和 Emoji 选择器。

高级图片编辑：

AI 去背景（集成 Remove.bg API 或 @imgly/background-removal）。

基础编辑（圆角、阴影、滤镜、亮度、对比度、镜像翻转）。

双击裁剪。

阶段 3: AI 能力与图库集成

AI 生图： 集成 Seedream 4.0 API，支持文生图。

AI 改图： 支持选中图片后通过提示词修改图片（例如，转为卡通风格）。

第三方图库： 集成 Unsplash API，用于搜索和使用高质量免费图片。

个人图库： 自动保存在AI生图和编辑过程中产生的图片。

阶段 4: 社区与分享

数据库后端： 启用 Vercel Postgres 数据库。

网页分享： 支持生成唯一网页链接（如 cover.qiaomu.ai/share/...）来分享设计作品。

模板共享： 用户可以将自己的设计"共享为模板"，供他人使用。

素材共享： 用户可以将自己的图片（如Meme图、透明图）"分享素材"，供他人使用。
