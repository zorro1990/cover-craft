## 界面介绍

### 整体布局

编辑器采用三栏布局设计：

```
┌─────────────────────────────────────────┐
│              顶部工具栏                    │
├──────────────┬──────────────┬────────────┤
│              │              │            │
│   左侧面板   │   画布区域   │  右侧面板  │
│              │              │            │
│  素材工具    │              │  属性编辑  │
│  ├文字      │              │            │
│  ├图片      │              │  选中对象  │
│  └形状      │              │  的属性    │
│              │              │            │
└──────────────┴──────────────┴────────────┘
```

### 左侧面板 - 素材工具

**文字工具**
- 添加文本对象
- 查看文字使用说明

**图片工具**
- 上传本地图片
- 查看支持格式

**形状工具**
- 绘制基本形状
- 查看快捷键

### 中间区域 - 画布

**画布**
- 作品创作区域
- 显示所有对象
- 支持拖拽操作

### 右侧面板 - 属性编辑

当选中对象时显示：
- 对象基本信息
- 属性编辑控件
- 操作按钮

## 基础操作

### 选择对象
- **单击对象**: 选中单个对象
- **显示控制框**: 选中后出现蓝色控制框
- **取消选择**: 单击空白区域

### 移动对象
1. 选中对象
2. 按住鼠标拖拽
3. 移动到目标位置释放

### 缩放对象
1. 选中对象显示控制点
2. 拖拽角部控制点
3. 保持 Shift 键锁定比例

### 旋转对象
1. 选中对象
2. 拖拽顶部旋转控制点
3. 旋转到目标角度释放

### 删除对象
**方法一**:
1. 选中对象
2. 按 `Delete` 键

**方法二**:
1. 选中对象
2. 在右侧属性面板点击"删除"按钮

## 文本编辑

### 创建文本
1. 点击左侧面板"文字"按钮
2. 点击"添加文字"按钮
3. 在画布上输入文字内容

### 编辑文本内容
1. 双击画布上的文字对象
2. 直接编辑文字
3. 点击空白处完成编辑

### 调整文字属性

**在右侧属性面板可以调整**:

- **字体**: 从下拉列表选择字体
- **大小**: 拖拽滑块或输入数值
- **颜色**: 点击颜色选择器
- **粗细**: 选择 Normal 或 Bold
- **斜体**: 切换 Italic 开关
- **对齐**: 选择 Left、Center 或 Right
- **不透明度**: 拖拽滑块调整透明度

**实时预览**: 所有修改会立即在画布上显示

## 图片编辑

### 上传图片
1. 点击左侧面板"图片"按钮
2. 点击"上传图片"按钮
3. 在文件选择对话框中选择图片
4. 等待上传完成

**支持的格式**:
- JPG (推荐)
- PNG
- WebP

**文件限制**:
- 最大文件大小: 10MB
- 自动压缩: 最大边不超过 800px

### 编辑图片属性

**在右侧属性面板可以查看**:

- **类型**: 显示为"图片"
- **原始尺寸**: 图片的原始大小
- **当前尺寸**: 图片的当前大小
- **文件大小**: 格式化显示的文件大小
- **缩放比例**: 当前缩放百分比

**可以调整的属性**:

- **不透明度**: 拖拽滑块 0-100%
- **旋转角度**: 拖拽滑块或使用快速按钮
  - 左转 90°: 逆时针旋转
  - 右转 90°: 顺时针旋转
- **翻转**:
  - 水平翻转: 左右镜像
  - 垂直翻转: 上下镜像
- **恢复原始尺寸**: 重置为 100% 缩放

### 图片变换
- **移动**: 拖拽图片
- **缩放**: 拖拽控制点
- **旋转**: 拖拽旋转控制点

## 形状绘制

### 绘制形状

**方法一: 使用按钮**
1. 点击左侧面板"形状"按钮
2. 选择要绘制的形状：
   - 矩形
   - 圆形
   - 直线
3. 形状自动添加到画布中心

**方法二: 使用快捷键**
- 按 `R` 键绘制矩形
- 按 `O` 键绘制圆形
- 按 `L` 键绘制直线

### 编辑形状属性

**形状信息**:
- **类型**: 矩形/圆形/直线
- **原始尺寸**: 形状的原始大小
- **当前尺寸**: 形状的当前大小
- **缩放比例**: 当前缩放百分比

**可以调整的属性**:

- **不透明度**: 滑块调整 0-100%
- **旋转角度**: 滑块或快速按钮调整
  - 左转 90°: 逆时针旋转
  - 右转 90°: 顺时针旋转
- **翻转**:
  - 水平翻转: 左右镜像
  - 垂直翻转: 上下镜像
- **边框宽度** (矩形、圆形):
  - 滑块调整 0-20px
  - 直线形状无此选项

### 形状变换
- **移动**: 拖拽形状
- **缩放**: 拖拽控制点（锁定纵横比）
- **旋转**: 拖拽旋转控制点

## 快捷键

### 形状绘制
| 键位 | 功能 | 说明 |
|------|------|------|
| `R` | 绘制矩形 | 快速创建矩形 |
| `O` | 绘制圆形 | 快速创建圆形 |
| `L` | 绘制直线 | 快速创建直线 |

### 通用操作
| 键位 | 功能 | 说明 |
|------|------|------|
| `Delete` | 删除对象 | 删除选中的对象 |
| `Ctrl/Cmd + C` | 复制 | 复制选中对象 |
| `Ctrl/Cmd + V` | 粘贴 | 粘贴复制对象 |

**注意**:
- 快捷键在编辑文本时不会触发
- 快捷键在输入框聚焦时不会触发

## 导出作品

### 复制到剪贴板
1. 点击顶部工具栏"复制"按钮
2. 作品自动复制到剪贴板
3. 可以直接粘贴到其他应用

### 下载作品
1. 点击顶部工具栏"下载"按钮
2. 选择保存位置
3. 文件格式为 PNG
4. 等待下载完成

## 常见问题

### Q: 如何更改画布大小？
**A**: 目前版本画布大小固定在创建时选择，暂时不支持动态更改。

### Q: 支持哪些图片格式？
**A**: 支持 JPG、PNG、WebP 三种格式。

### Q: 图片文件大小有限制吗？
**A**: 单个图片文件最大支持 10MB，超过会自动压缩。

### Q: 如何撤销操作？
**A**: 目前版本暂未实现撤销功能，请谨慎操作。

### Q: 可以同时选择多个对象吗？
**A**: 目前版本不支持多选，每次只能操作一个对象。

### Q: 如何调整对象的层级？
**A**: 目前版本暂未实现图层管理功能。

### Q: 导出的图片分辨率是多少？
**A**: 导出的图片分辨率与画布设置一致，保持高清质量。

### Q: 快捷键不生效怎么办？
**A**: 请确认当前没有在输入框中输入文字，快捷键在文本编辑时不会触发。

### Q: 如何恢复对象的原始大小？
**A**: 选中对象后，在右侧属性面板点击"恢复原始尺寸"按钮。

### Q: 作品可以保存到云端吗？
**A**: 目前版本暂未实现云端保存功能，建议及时导出作品。
