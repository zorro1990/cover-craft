# Phase 2 QA 测试用例

## 测试用例 6：高级字体

### 用例 ID: TC-006
**测试目标**: 验证用户可以使用精选中文字体

**前置条件**: 已添加文字到画布

**测试步骤**:
1. 选中画布上的文字元素
2. 打开右侧属性面板
3. 点击字体选择器
4. 浏览可用的中文字体列表
5. 选择一种字体
6. 检查文字外观变化

**预期结果**:
- 字体选择器正常显示
- 至少显示 18 种精选字体
- 文字外观根据选择的字体改变
- 字体加载速度合理

**测试状态**: [ 待测试 ]
