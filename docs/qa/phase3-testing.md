# Phase 3 QA 测试用例

## 测试用例 11：AI 生图

### 用例 ID: TC-011
**测试目标**: 验证用户可以使用 Seedream API 生成图片

**前置条件**: 用户已配置 Seedream API Key

**测试步骤**:
1. 点击左侧面板"AI生图"
2. 打开提示词输入框
3. 输入描述："一只可爱的橘猫"
4. 点击"生成"按钮
5. 等待 API 返回结果
6. 在面板中查看生成的图片
7. 点击图片将其添加到画布

**预期结果**:
- AI生图面板正常打开
- 提示词输入框正常显示
- API 调用成功
- 生成的图片质量合理
- 图片可以成功添加到画布
- 生成的图片保存到个人图库

**测试状态**: [ 待测试 ]

## 测试用例 12：AI 改图

### 用例 ID: TC-012
**测试目标**: 验证用户可以使用 AI 修改图片风格

**前置条件**: 画布上已添加图片

**测试步骤**:
1. 选中图片
2. 在右侧面板点击"AI改图"
3. 打开提示词输入框
4. 输入风格描述："转换为卡通风格"
5. 点击"应用"按钮
6. 等待处理完成
7. 检查图片风格变化

**预期结果**:
- AI改图选项正常显示
- 提示词输入框正常
- API 调用成功
- 图片风格正确改变

- 处理时间在合理范围内

## 测试用例 13：Unsplash 图库搜索

### 用例 ID: TC-013
**测试目标**: 验证用户可以搜索和使用 Unsplash 图库

**前置条件**: 用户已配置 Unsplash API Key

**测试步骤**:
1. 点击左侧面板"图库"
2. 选择"Unsplash"选项
3. 在搜索框输入"nature"
4. 查看搜索结果
5. 筛选搜索结果（可选）
6. 点击选择图片
7. 图片添加到画布

**预期结果**:
- Unsplash 图库正常打开
- 搜索功能工作正常
- 显示高质量图片结果
- 分页加载正常
- 图片可以添加到画布
- 正确显示图片来源信息

## 测试用例 14：个人图库管理
