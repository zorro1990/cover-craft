# Phase 4 QA 测试用例

## 测试用例 15：网页分享

### 用例 ID: TC-015
**测试目标**: 验证用户可以生成唯一 URL 分享设计稿

**前置条件**: Vercel Postgres 数据库已配置

**测试步骤**:
1. 在画布上创建设计内容
2. 点击顶部工具栏"分享"
3. 生成唯一分享链接
4. 复制链接
5. 在新浏览器窗口打开链接
6. 检查分享页面显示
7. 检查设计内容完整性

**预期结果**:
- 分享功能正常工作
- 成功生成唯一 URL
- 链接可以正常复制
- 分享页面正常加载
- 设计内容完整显示
- URL 格式正确（如 cover.qiaomu.ai/share/DVlejG）

**测试状态**: [ 待测试 ]

## 测试用例 16：模板共享

### 用例 ID: TC-016
**测试目标**: 验证用户可以共享设计为模板

**前置条件**: Vercel Postgres 数据库已配置

**测试步骤**:
1. 创建满意的设计稿
2. 点击顶部工具栏"另存为模板"
3. 弹出窗口，输入模板名称
4. 设置为公开模板
5. 点击"共享"
6. 检查成功提示
7. 切换用户账号
8. 打开左侧面板"模板"
9. 浏览共享模板列表
10. 点击选择模板
11. 模板加载到画布

**预期结果**:
- 另存为模板功能正常
- 模板名称输入正常
- 模板成功保存到数据库
- 其他用户可以浏览模板
- 模板列表正常显示
- 模板可以成功加载
- 加载的模板保持原始设计

**测试状态**: [ 待测试 ]

## 测试用例 17：素材共享

### 用例 ID: TC-017
**测试目标**: 验证用户可以共享图片为素材

**前置条件**: Vercel Postgres 数据库已配置

**测试步骤**:

2. 选中图片
3. 点击"共享为素材"
4. 设置素材类型（Meme/透明图/其他）
5. 添加搜索标签
6. 设置为公开

8. 切换用户账号
9. 浏览共享素材库
10. 搜索特定素材
11. 选择并使用素材

- 共享为素材功能正常
- 素材类型设置正常
- 标签可以正常添加
- 素材成功保存到数据库
- 其他用户可以浏览素材

## 测试用例 18：数据库操作性能

### 用例 ID: TC-018
**测试目标**: 验证数据库操作的性能和稳定性

**前置条件**: Vercel Postgres 数据库已配置

**测试步骤**:
1. 连续保存 10 个设计稿
2. 连续分享 10 个模板
3. 连续共享 10 个素材
4. 监控数据库响应时间
5. 检查数据库连接稳定性

**预期结果**:
- 数据库操作响应时间 < 2 秒
- 连接稳定性良好
- 无数据丢失
- 无并发冲突

**测试状态**: [ 待测试 ]
