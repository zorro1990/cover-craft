# MVP 阶段 QA 测试用例

## 测试用例 1：画布创建

### 用例 ID: TC-001
**测试目标**: 验证用户可以创建指定尺寸的画布

**前置条件**: 用户访问首页

**测试步骤**:
1. 打开 cover.qiaomu.ai
2. 检查是否显示默认画布
3. 在顶部工具栏选择画布尺寸选项
4. 选择 3:4 比例
5. 选择 1:1 比例
6. 选择 4:3 比例

**预期结果**:
- 画布默认显示
- 尺寸选项可以正常选择
- 画布尺寸应该根据选择调整

**测试状态**: [ 待测试 ]

## 测试用例 2：添加文字

### 用例 ID: TC-002
**测试目标**: 验证用户可以在画布上添加和编辑文字

**前置条件**: 画布已创建

**测试步骤**:
1. 点击左侧面板"文字"工具
2. 在画布上点击添加文本框
3. 输入测试文字
4. 在右侧属性面板修改字体
5. 修改字体大小
6. 修改文字颜色
7. 修改文字背景色

**预期结果**:
- 文字工具可以正常点击
- 文本框成功添加到画布
- 文字内容可以编辑
- 字体、大小、颜色和背景色可以修改

**测试状态**: [ 待测试 ]

- 图片可以拖拽移动

### 用例 ID: TC-004

**前置条件**: 画布已创建

**测试步骤**:
1. 在画布上按 R 键绘制矩形

4. 选中矩形，拖拽调整大小
5. 选中圆形，拖拽调整大小
6. 选中直线，拖拽调整长度和角度

**预期结果**:
- 快捷键可以成功创建对应形状
